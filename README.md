# Задание 1

## Задача 1

### Дерево содержит строки. В конец каждой строки приписать ее же. 

### Алгоритм решения

1)Заполнение дерева через функцию FillTree
  Строки формируются случайно через функцию RandString
    В функции RandString случайно генерируется индекс от нуля до длины алфавита и выбирается элемент из алфавита с этим индексом, добавляется в строку
2)Вывод дерева через фукнцию Print
3)Создание нового дерева с измененными элементами через функцию MapTree
  в аргументах этой функции аргументы f - функция, которая будет изменять элемент для нового дерева, tree - изначальное дерево
  в функции conc происходит конкатенация строки s
4)Вывод нового дерева с измененнными элементами

# Задание 2

## Задача 1

### Текст задачи

Выяснить, содержится ли в дереве элемент с заданным значением.  

### Алгоритм решения

1)Заполнение дерева через функцию FillTree
  Строки формируются случайно через функцию RandString
    В функции RandString случайно генерируется индекс от нуля до длины алфавита и выбирается элемент из алфавита с этим индексом, добавляется в строку
2)Вывод дерева через фукнцию Print
3)Затем пользователь вводит элемент
4)Задается значение res, обозначающее содержание элемента в дереве, с помощью функции Contains (element - введенный элемент, tree - дерево, в котором будет производится поиск)
  Folder - функция для проверки нахождения элемента. Если acc - true, то возвращается true, иначе nodeValue = value 
  FoldTree - рекурсивная функция, которая проходит по дереву и применяет функцию Folder
5) если res = true, то выводится сообщение о наличии элемента
  иначе выводится сообщение, что элемента нет в дереве
